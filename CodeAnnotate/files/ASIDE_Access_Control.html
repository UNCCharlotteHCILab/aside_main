The line has Access Control Vulnerability:

+ Why am I seeing this page?

ASIDE has detected that your code contains a call to a function that accesses secure information. This does not necessarily mean that this code is vulnerable; ASIDE is just asking you to identify where the access control check takes place.

+ What is an access control vulnerability?

An access control vulnerability exists when a real, authorized user has access to data or functions that they should not have access to. For example, a if a regular forum user was able to see or modify another user's private profile information, that would be an access control vulnerability.

+ What can I do to fix it??

Simply highlight the access control check for the sensitive function! You can do this by clicking on the yellow question mark, then double-clicking on the 'Annotate Now' option, and highlighting the relevant check.
